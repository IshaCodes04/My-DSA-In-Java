*Data types in java*

- There are 2 types of data types in java 

1. Primitive data type 

- Primitive means those data types which already exists in java for examples :- 

- byte = stores only 256 numbers
- short
- char = only single character are stored
- boolean = true or false will be stored
- int
- long
- float
- double

2. Non-primitve data type

- Non-primtive means thoda data types which created by themselves for use for example :-

- String
- Array
- Class
- Object 
- Interface
